# FX_RISK
USD/KRW FX Risk — Regime Detection & Corporate Hedging Signal
원/달러 환율 변동 리스크를 주제로, 기존의 예측 중심론에서 벗어난 리스크 관리 프레임워크를 제안하고자 한다.
기업이 달러 자산을 보유하는 순간, 환율 리스크는 피할 수 없다.
그러나 효율적 시장 가설 하에 따라 내일의 환율을 예측하는 것은 사실상 불가능에 가깝기 때문에 실무자들이 주로 마주하는 질문은 "환율이 얼마가 될까?"보다, "지금 헤지 비중을 어떻게 둬야할까?"에 가깝다.

main theme : USD/KRW 환율 리스크를 가격 예측이 아닌, 상태 인식(state detection) 문제로 재정의한 분석
Hidden Markov Model을 활용하여 시장을 세 개의 레짐으로 나누고, 각 레짐을 기업의 헤지 의사결정 신호로 연결하였다.
분석 대상은 2000년 1월 1일부터 2026년 5월까지 USD/KRW 일별 데이터(거래일 5,761일), 금융 데이터 분석 / 리스크 관리 포트폴리오 제작에 포커싱하여 설계된 논리 구조이다.


다루는 질문

1. KRW 시장은 통계적으로 구분되는 몇 개의 상태로 나뉘는가
2. 각 상태의 손실 분포는 어떻게 다른가
3. 상태를 탐지했을 때, 기업은 실제로 행동할 시간이 있는가

세 번째 질문이 이 프로젝트의 핵심이다. 대부분의 레짐 분석은 "탐지하면 대응하라"에서 멈추지만, 탐지에서 실행까지 걸리는 시간을 실측하면 그 처방이 충격 유형에 따라 무너진다는 것을 보인다.
